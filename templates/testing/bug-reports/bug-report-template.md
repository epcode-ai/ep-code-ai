**标题示例**:

## 详细信息

### 描述

[用 2-3 句话描述问题]

### 严重级别

- [ ] **P0 阻塞**：核心功能不可用、数据丢失、安全漏洞
- [ ] **P1 严重**：主要功能异常、影响业务流程
- [ ] **P2 一般**：次要功能异常、有 workaround
- [ ] **P3 轻微**：UI/文案、优化建议

### 优先级

- [ ] **高**：本版本必须修复
- [ ] **中**：本版本尽量修复
- [ ] **低**：可延后修复
