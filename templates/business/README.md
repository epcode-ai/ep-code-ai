# 业务场景模板库

> 配合 [业务篇](../../docs/chapters/02-business/) 使用的现成模板

## 模板清单

| 模板 | 用途 | 何时使用 |
|------|------|---------|
| [PRD 模板](./prd-template.md) | 产品需求文档 | 每个新需求立项后 |
| [用户故事模板](./user-story-template.md) | 用户故事 + 验收标准 | PRD 拆解阶段 |
| [业务规则模板](./business-rule-template.md) | 决策表 + 状态机 | 复杂业务规则梳理 |
| [变更请求（CR）模板](./change-request-template.md) | 需求变更单 | 项目中途变更时 |
| [竞品分析模板](./competitor-analysis-template.md) | 竞品调研 | 新功能立项前 |

## 使用方式

1. 复制整个文件到你的项目 Wiki / Confluence / 仓库 `docs/` 目录
2. 替换 `[xxx]` 为实际内容
3. 根据项目情况裁剪章节（不是每次都全用）

## 定制建议

**小项目**：PRD 模板可以简化到只保留必要章节。
**大项目**：所有模板全用，并做跨角色评审。
**跨部门协作**：PRD + CR 是基础配置。
