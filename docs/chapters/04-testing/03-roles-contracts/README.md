## 契约的四个核心要素

每份契约都必须明确：

1. **交付物**：必须提交什么（附模板链接）
2. **时限**：什么时候交
3. **质量标准**：交付物的验收标准
4. **违约处理**：不达标怎么办

## 通用原则

- **对事不对人**：违约是流程问题，不是人品问题
- **首次警告，二次上报**：保留记录，每月统计
- **契约写进流程文档**：让全员知晓，避免"我不知道"
- **定期复盘**：每个版本结束后评估契约执行情况
