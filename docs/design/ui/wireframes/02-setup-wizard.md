## 关键交互

| 步骤 | 行为 |
|------|------|
| 任意步 | 右上 `✕ 跳过引导`: 标记 setup 状态为"未完成",下次启动仍提示 |
| 步骤 2 | 一键安装按钮内嵌 Claude Code 安装命令,输出实时日志在折叠区 |
| 步骤 2 | "下一步" 在有 ❌ 时禁用;有 ⚠️ 但无 ❌ 时可继续(警告 toast) |
| 步骤 3 | API Key 输入框默认遮蔽,👁 切显隐;支持粘贴 |
| 步骤 3 | 测试连接前 "下一步" 禁用,确保 API Key 有效后才放行 |
| 步骤 4 | 选 "创建第一个会话" 跳到主视图并自动 + 新会话 |

## 跳过的影响

- **步骤 2 跳过**: 主视图状态栏显示 ⚠️ 红色 "环境未配置",点击重开向导
- **步骤 3 跳过**: 主视图无法发消息,顶栏供应商显示 "未配置 ⚠️",点击跳到配置
- **步骤 4 跳过**: 直接进主视图,行为与"进入应用"等同
