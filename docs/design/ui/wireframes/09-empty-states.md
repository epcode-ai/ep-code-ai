## 关键原则

| ❌ 不要 | ✅ 要 |
|--------|------|
| 只显示 "暂无数据" | 显示**为什么没有 + 下一步做什么** |
| 用大图标占满屏 | 适度图标 + 明确 CTA |
| 把空状态做成营销页 | 保持简洁,1-2 个核心 CTA + 1-2 个备选路径 |
