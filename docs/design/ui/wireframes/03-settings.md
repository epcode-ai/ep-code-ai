## 关键交互

- 切换 tab 时不丢已修改未保存的内容(顶部 banner 提示"有未保存更改")
- "保存" 按钮在有改动时高亮,无改动时灰色
- "取消" 关闭并丢弃改动,有改动时弹确认框
- 任何会影响连接的更改(API Key、Endpoint)保存后自动重测连接
- "清空全部本地数据" 二次确认要求输入 `DELETE` 确认
