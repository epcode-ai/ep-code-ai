## 关键交互

| 元素 | 行为 |
|------|------|
| ◀ 返回会话 | 关闭全屏,回到主视图三栏布局 |
| 📋 复制 | 复制全文到剪贴板 |
| 💾 导出 | 弹格式选择: `.md`(原始) / `.pdf`(渲染后) / `.html`(嵌入样式) |
| ◀▶ 切换 Artifact | 在本会话所有 Artifact 间循环切换 |
| 版本下拉 | 选历史版本看,选 "对比 v2 ↔ v3" 进入对比模式 |
| 大纲 | 点击跳到对应行,鼠标 hover 显示行号 |
| 📍 跳转到生成时对话 | 关全屏 + 滚动聊天到生成这个 artifact 的那条 AI 回复 |
| 📤 发送到 Jira | 弹 Jira 集成对话框(Project / Type / Summary 预填) |
| ✨ AI 重新润色 | 在聊天里插入 prompt "请优化以下 Artifact: ..." 并附带 |
| 合并(对比模式) | 选 "用 v3 替换 v2 / 选区合并 / 创建新版本" |

## 大纲生成

- Markdown: 自动从 `#`-`####` 抽取
- JSON / YAML: 抽顶层 keys
- 代码: 抽 function / class / export
- 其他: 显示 "无大纲"

## 元数据

| 字段 | 来源 |
|------|------|
| 生成 | AI 第一次输出这个 artifact 的时间 |
| 修订 | 后续 AI 改/用户改的次数 |
| 来源 | 会话名 + 链接(点击关全屏跳到对话) |
