## 关键交互

| 元素 | 行为 |
|------|------|
| 当前供应商旁的 ● | 不可选(已选中) |
| 状态徽章 | 实时反映上次连接结果。点击徽章重测连接 |
| 默认 "在新会话中切换" | 安全选项,90% 用户用这个 |
| "在当前会话中切换" | 进入二次确认,要求输入 "切换" 确认 |
| esc | 关浮层,无操作 |
| 滚轮 / ↑↓ | 在可切换列表中导航 |
| ↵ | 按勾选项执行 |

## 切换后的视觉反馈

- 顶栏供应商名 0.5s 淡入淡出更新
- 中栏标题更新为新供应商 + 模型
- 在聊天流插入 "⚙ 11:45 · 已切换到 AWS Bedrock" 系统消息
