## 元素说明

| # | 元素 | 行为 |
|---|------|------|
| ① | 顶栏 · 供应商切换 | 点 `[Anthropic ▾]` 弹切换浮层(见 wireframe-06)。`⚙` 开设置(wireframe-03) |
| ② | 中栏标题区 | 显示当前会话名 + 模型 + token 用量。`📌` 收藏切换;`⋮` 弹更多操作(改名/导出/删) |
| ③ | 右栏 Artifacts | 列出本会话产生的所有 Artifact。可折叠/展开。无 Artifact 时整个右栏自动折起 |
| ④ | 底部输入栏 | `/` 触发命令面板(wireframe-04);`📎` 附件;`⚡` 切模型(本会话内);`⌘↵` 发送 |
| ⑤ | 状态栏 | 持久显示环境健康 + 网络状态(wireframe-10) |

## 三栏宽度

- 左侧 240px(可在 200-360 拖拽,记忆)
- 右侧 320px(可在 280-500 拖拽;无 Artifact 自动 0px)
- 中间 flex(min 480px,小于则强制折叠左/右)

## 响应式

| 窗口宽度 | 行为 |
|---------|------|
| `≥ 1280` | 三栏全开 |
| `1024 - 1280` | 右栏自动折叠成抽屉(图标按钮唤起) |
| `768 - 1024` | 左栏也折叠,变成移动版双层抽屉 |
| `< 768` | (不支持,强制最小窗口 768) |

## 会话列表分组

按以下顺序固定:
1. **收藏**(⭐ 标记的,任何时候置顶)
2. **今天 / 昨天 / 本周** (按最后活动时间)
3. **本月 / 更早** (按月分组)

每组可折叠。计数显示在组名旁。

## 空状态

- 无会话: 中栏显示欢迎卡 + "+ 新会话 ⌘N" 大按钮
- 无 Artifact: 右栏完全折叠,不显示空白
- 详见 wireframe-09
