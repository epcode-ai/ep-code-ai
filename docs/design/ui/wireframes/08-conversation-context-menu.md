> 侧边栏会话右键 / macOS Ctrl+点击 / 长按(触摸板)触发。

## 重命名内联编辑

```
   │ ▼ 今天 (3)        │
   │ ┌──────────────┐  │
   │ │需求评审 - 离职│  ← 输入框获得焦点,蓝色边框
   │ └──────────────┘  │
   │ ○ Bug-1024        │      ↵ 保存   esc 取消
```

## 收藏与置顶

收藏的会话有独立 "⭐ 收藏" 分组,任何时候置顶,顺序按收藏时间倒序:

```
   │ ▼ 收藏 ⭐ (2)      │
   │ ⭐ 故障复盘 - DB 慢 │  ← 收藏图标 + 始终在最上
   │ ⭐ ADR 模板         │
   │ ─────────         │
   │ ▼ 今天 (3)         │
   │ ...                │
```

置顶(📌)只在所在分组内置顶,不跨分组:

```
   │ ▼ 今天 (3)         │
   │ 📌 紧急 Bug 排查    │  ← 置顶,在"今天"组首位
   │ ○ 需求评审          │
   │ ○ 部署 SOP          │
```

## 分组管理

新建分组弹小窗:

```
                ┌─────────────────────────────┐
                │  新建分组                     │
                ├─────────────────────────────┤
                │  名称  [____________________]│
                │  颜色  ●  ●  ●  ●  ●  ●     │
                │        红 橙 黄 绿 蓝 紫     │
                │                              │
                │  取消              创建      │
                └─────────────────────────────┘
```

分组在侧边栏的视觉:

```
   │ ▼ 工作 ● (5)       │  ← 分组色点
   │ ○ 需求评审          │
   │ ○ Bug-1024          │
   │ ...                 │
   │ ▼ 学习 ● (2)       │
   │ ○ Rust 入门         │
```

## 多选(⌘ 点击 / Shift 点击)

```
   │ ▼ 今天 (3)         │
   │ ☑ 需求评审    🔵   │  ← 多选,蓝色高亮
   │ ☑ Bug-1024    🔵   │
   │ ○ 部署 SOP         │
   │                    │
   │ ─────────         │
   │ 📦 已选 2 项 [操作 ▾] │  ← 底部操作条
   │   • 移到分组         │
   │   • 加入收藏         │
   │   • 批量导出         │
   │   • 删除选中         │
```

## 删除二次确认

```
                ┌─────────────────────────────┐
                │  ⚠️  删除会话                  │
                ├─────────────────────────────┤
                │                              │
                │  会话「需求评审 - 离职流程」  │
                │  和它的 6 条消息将被删除      │
                │                              │
                │  Artifact 会一起删除吗?       │
                │  ● 一起删 (推荐)              │
                │  ○ 保留 Artifact (移到回收站)  │
                │                              │
                │  取消               删除      │
                └─────────────────────────────┘
```

## 关键交互

- 删除有 30 天回收站(设置可关)
- 移到分组支持拖拽
- 重命名支持 Tab 自动补全(基于会话内容生成的候选)
