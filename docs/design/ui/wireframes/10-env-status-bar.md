# Wireframe 10 · 持久化环境健康状态栏

> 永远在主视图最底部 24px 高度。一眼能看出环境健康 + 网络状态。

## 健康状态(全绿)

```
├──────────────────────────────────────────────────────────────────────────────────┤
│  ✅ 环境正常 · Node v20 / Claude Code v1.2.3        ●  Anthropic · 240ms          │
└──────────────────────────────────────────────────────────────────────────────────┘
   ↑                                                  ↑
   左半 · 环境信息                                    右半 · 网络/供应商
```

## 异常状态(环境某项失败)

```
│  ⚠️ Claude Code 缺失 [▶ 修复]                       ●  Anthropic · 240ms          │
   ↑ 黄色高亮                                          ↑ 网络仍正常
```

## 严重异常(全红)

```
│  ❌ 环境损坏 [▶ 修复]                                ●  断网 [🔄 重试]              │
   ↑ 红色背景                                          ↑ 红色
```

## 加载中状态

```
│  ⏳ 检查环境中...                                    ⏳ 测连接中...                  │
```

鼠标 hover 在左半,弹 tooltip 显示完整列表:

              ↑

## 点击行为

| 点击区域 | 行为 |
|---------|------|
| 左半 (环境) | 异常时弹模态修复对话框;正常时打开"环境检查"页 |
| `[▶ 修复]` 按钮 | 直接开始一键修复(走 wizard 步骤 2 的逻辑) |
| 右半 (供应商) | 弹供应商切换浮层(wireframe-06) |
| `[🔄 重试]` | 立即重测连接 |

## 视觉规范

- **高度**: 24px(固定)
- **左右内边距**: 16px
- **背景**: 默认与窗口背景同;异常时变 `Warning` / `Error` 浅色背景
- **字号**: 12px
- **图标**: 用 emoji,跨平台一致

## 不显示状态栏的场景

- 全屏 Artifact 详情(wireframe-05) 时隐藏
- 设置页(wireframe-03) 时隐藏
- 首次向导(wireframe-02) 时隐藏

## 通知降级

如果系统通知(macOS Notification Center / Windows Toast / Linux D-Bus)被禁:
- 严重错误从静默升级为底部 banner(状态栏上方插入,可关闭)

```
│  ⚠️ Claude Code 进程崩溃,正在重启...    [详情] [✕]                                  │
├──────────────────────────────────────────────────────────────────────────────────┤
│  ❌ 环境损坏 [▶ 修复]                               ●  断网 [🔄 重试]              │
└──────────────────────────────────────────────────────────────────────────────────┘
```
