## 关键交互

| 触发条件 | 行为 |
|---------|------|
| 输入框光标在第一位输入 `/` | 立即弹面板,无延迟 |
| 输入框中间输入 `/` | **不**触发(避免误触) |
| 选中后按 ↵ | 如有必填参数,进参数补齐;否则直接执行 |
| 参数补齐界面按 ⌘↵ | 直接执行(跳过 "✓ 执行" 按钮) |
| 任何时刻按 esc | 关面板,光标回到输入框,保留已输入的 `/xxx` 文本 |
| 按 → 或点 "[→ 帮助]" | 在聊天流插入命令的完整 Markdown 帮助 |

## 命令来源

命令面板的清单 = `tools/cli/commands/` 目录扫描结果。新加 CLI 子命令自动出现,无需 GUI 改动。

每个命令的元数据(描述 / 参数定义)从对应 `.js` 文件的 JSDoc 头部提取。
