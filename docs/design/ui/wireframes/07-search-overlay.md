## 筛选模式(用户按 ⌃→ 或点筛选)

```
        ┌──────────────────────────────────────────────────────────┐
        │  🔍  PRD              筛选: [✓会话 ✓Artifact ✗消息 ✓命令]  │
        ├──────────────────────────────────────────────────────────┤
        │  时间  ●全部  ○今天  ○本周  ○本月  ○自定义                 │
        │  供应商 [全部 ▾]                                            │
        ├──────────────────────────────────────────────────────────┤
        │  ...(搜索结果,按筛选条件过滤)...                           │
        └──────────────────────────────────────────────────────────┘
```

## 关键交互

| 元素 | 行为 |
|------|------|
| 输入框 | 输入即搜,300ms 防抖。空时显示 "最近访问 + 常用命令" |
| 类型分组 | 默认顺序 会话 → Artifact → 命令 → 消息;每组最多 3-5 条,有更多时显示 "[展开 →]" |
| ↑↓ | 跨组导航 |
| ↵ | 打开高亮项 |
| ⌃→ | 进/退筛选模式 |
| 搜索消息 | 显示消息片段(关键词高亮),点击跳到该消息位置并滚动 |
| esc | 关浮层 |
| Cmd+1/2/3/4 | 直接跳到第 N 项 |

## 搜索算法

- 会话/Artifact 名: 子串匹配 + 模糊匹配
- 消息内容: 全文索引(SQLite FTS5)
- 命令: 命令名 + 描述模糊匹配

## 性能

- 本地索引,首次启动建索引(后台任务,UI 不阻塞)
- 输入到结果出现 < 100ms (10000 条会话规模)
