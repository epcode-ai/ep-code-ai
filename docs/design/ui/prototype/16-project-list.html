💡 什么是"接入模式"?

不同阶段的项目接入本框架的路径不同。这是本框架的一等公民抽象,帮你判断"从哪儿开始、先做什么、别纠结什么"。

A · 绿地

刚立项、从 0 开始的项目。全套上车。

B · 开发中

代码已写未上线。两条腿走路,从今天规范。

C · 迭代中

已上线定期迭代。每版本加一层,5 迭代走完。

D · 稳态运维

运行多年只做维护。聚焦运维场景。

👉 详细判定 + 起步清单: docs/chapters/00-adoption/
