## 与 macOS 的差异（预览）

| 项目 | macOS | Linux |
|------|-------|-------|
| 包管理器 | Homebrew | apt / yum / dnf / pacman |
| 用户主目录 | `/Users/<name>` | `/home/<name>` |
| 文件系统 | 默认大小写不敏感 | 大小写敏感 |
| 默认 shell | zsh | bash |
| 行尾符 | LF | LF |
