### 2.5 性能测试结果

| 接口 | 目标 P99 | 实测 P99 | QPS 目标 | QPS 实测 | 结果 |

| POST /leaves/:id/approve | 500ms | 298ms | 100 | 142 | ✅ |
| GET /leaves/balance | 100ms | 48ms | 1000 | 2400 | ✅ |
| 导出 3k 行 | 10s | 3.8s | - | - | ✅ |
| 导出 10k 行 | 10s | 14.2s | - | - | ⚠️ **未达标** |

**未达标处置**:
- 已知问题,计划在 v1.1 优化（流式 + 后台任务模式）
- 对 HR 的应对: 培训时说明建议单次导出 ≤ 5000 行
- 风险: 低（HR 月度数据 ~300 行，少有超过 5000 行场景）
